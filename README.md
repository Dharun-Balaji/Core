# Core

[![Not AI Generated](https://img.shields.io/badge/Not-AI%20Generated-blue)](https://github.com/not-ai/not-ai)

Core is a high-performance, minimalist Kanban board designed for streamlined project management. Built with React, Redux, and TypeScript, it focuses on speed, reliability, and a premium user experience.

## Features

- **Dynamic Kanban Board**: Effortlessly manage tasks across multiple columns.
- **Real-time Updates**: State management powered by Redux Toolkit for a responsive UI.
- **Drag-and-Drop**: Intuitive task organization (powered by @dnd-kit).
- **Persistent Storage**: Your data stays with you, stored locally for instant access.
- **Premium Design**: A sleek, dark-themed interface built with modern CSS.

## Tech Stack

- **Frontend**: React 19
- **State Management**: Redux Toolkit
- **Icons**: Lucide React
- **Styling**: Vanilla CSS with modern variables
- **Build Tool**: Vite

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Dharun-Balaji/Core.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
